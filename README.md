# Global Sales Data Analysis — Pure Python
## Overview
Analyzed a global sales dataset sourced from Kaggle containing 100 records across multiple regions and product categories. This project was completed using pure Python with no external libraries like Pandas.
Objective
The goal was to extract meaningful insights from raw sales data using only Python fundamentals .
## Questions Answered

What unique regions and product categories exist in the dataset?

What was the maximum, minimum, and average units sold per order?

What was the maximum profit across all orders?

What was the price range of products sold?

### Key Findings

Total records analyzed: 100

Maximum units sold in a single order: 9,925

Minimum units sold in a single order: 124

Total units sold across all orders: 512,871

Maximum unit price: $668.27

Minimum unit price: $9.33

Total revenue: $137,348,768.31

Unique regions: 7 (Asia, Europe, Africa, North America, etc.)

Unique product categories: 11 (Fruits, Household, Cosmetics, etc.)


### Tools Used

Python 
Jupyter Notebook
Dataset: Sales CSV — https://www.kaggle.com/datasets/anairamcosta/sales-csv?resource=download

How to Run

Clone this repository

Download sales.csv from Kaggle and place it in the same folder

Open sales_analysis.ipynb in Jupyter Notebook

Run all cells from top to bottom
